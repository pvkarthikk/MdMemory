# Python Functions

Functions are reusable blocks of code that perform specific tasks.

## Syntax
```python
def function_name(parameters):
    # function body
    return value
```

## Key Points
- Use `def` keyword to define functions
- Parameters are optional
- Functions return values using `return` statement
